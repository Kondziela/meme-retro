# meme-retro
We would like to create Retroboard with Memes

Current phase: Creation Conception & Design

Join us and help to create life more Funny!!!

How to start:
 - install NodeJS
 - install npm
 - install gulp https://gulpjs.com/
 - call: npm install 
 - run: gulp
 - open: localhost:4000
